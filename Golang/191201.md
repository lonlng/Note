| verb     | 描述                                      |
| -------- | ----------------------------------------- |
| %d       | 十进制整数                                |
| %x,%o,%b | 十六进制,八进制,二进制数                  |
| %f,%g,%e | 浮点数:如3.1415926,3.1415926,3.141593e+00 |
| %t       | 布尔类型:true 或 false                    |
| %c       | 字符(Unicode)                             |
| %s       | 字符串                                    |
| %q       | 带引号字符串(如 "abc") 或者字符 (如 'c')  |
| %v       | 内置格式的任何值                          |
| %T       | 任何值的类型                              |
| %%       | 百分号本身(误操作数)                      |

`Printf` 默认不写换行符
